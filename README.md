# 📚 Elaboración de Documentación de Sistemas Informáticos

### Repositorio académico para la planificación, diseño y elaboración de documentación técnica de sistemas informáticos.

---

# 📖 Descripción

Este repositorio reúne las actividades, investigaciones, plantillas y documentos desarrollados durante la asignatura **Elaboración de Documentación de Sistemas Informáticos** del **Bachillerato Técnico Vocacional en Desarrollo de Software**.

El propósito principal es aplicar metodologías, estándares internacionales y buenas prácticas para la creación de documentación técnica y manuales de usuario, garantizando que la información sea clara, organizada y comprensible para diferentes tipos de usuarios.

---

# 🎯 Objetivos

- Elaborar documentación técnica profesional.
- Diseñar manuales de usuario claros y estructurados.
- Aplicar normas internacionales para documentación de software.
- Analizar herramientas utilizadas en la elaboración de documentación.
- Implementar plantillas reutilizables para futuros proyectos.
- Facilitar la comprensión y mantenimiento de sistemas informáticos.
